HTML & CSS

  • HTML & CSS are both "languages"
  • HTML controls the "structure" of the page
  • CSS controls the "style" of the page
  • When programmers talk about "DOM" (Document Object Model) they talk about the tree-like structure of a page

Recap with definitions

HTML (Hyperlink Markup Language) is a standard markup language to create web pages
CSS (Cascade Stayle Sheets) is a language used for describing the look and formatting of a document written in a markup language
DOM (Document Oject Model) is a cross platform and a language independent convention for representing & interacting with objects in HTML. The nodes of every documents are organized in a tre structure called the DOM tree.

Lesson 3

CSS: code written to control the "style" of HTML elements and avoid repetitions

Avoiding repetition
A web page is made of many elements that will appear similar, like headings, background-color, content structures of boxes, etc.
It would be very tedious and time-consuming to repeat the attribute of an element each time it is met in the HTML coding
To avoid these repetition, all common attributes of repetitive elements will be defined in a CSS file as described below

CSS language is built as:

selector {declaration;
}

  • everything between { and } will be interpreted as a rule applying to all elements selected by the selector (example h1)
  • a declaration will be in the form:

property:_value; (for example: background-color:_black;)

Use class selector to apply style to certain boxes
as in
"<"div class= "stop">
".stop {
"background-color: red;
"}
It will attribute the background color RED to all "stop" boxes.

Inheritance
Most of declarations will inherit, i.e. apply to all div within a "<"style">.

Include CSS styling in a web page

Method 1
Write CSS code in the "<"head"> of HTML.
Good for small projects.

Method 2
Link HTML to a separate CSS file:

  1. Write structural HTML in a file (main.html)
  2. Write all CSS code in another file (main.css)
  3. add "<"head"> to top of HTML
  4. add "<"link"> inside the "<"head">..."<"/head">(remember that link is a void tag)
  5. add atribute to link.

Code comments
To comment code:
in HTML: "<"!--comment--">
in CSS: /*---*/.

The concept of Boxes
Boxes have:
- content
- padding
- border
- margin
Each of those can be formatted (size, color, etc.)

The declaration "box-sizing:_border-box;" makes calculation of box size.

Sizing can be expressed in pixels (px) or in %.

Box positioning

  1. Div's are block elements (as opposed to inline): by default, they take up the entire page width
  2. Adding the rule "display:_flex;" to the appropriate CSS will override this behavior and let div's appear next to each other.
